
## 关于laravel-adminlte

laravel-adminlte 通过轻量级前端框架adminLte和laravel框架搭建的后台模版，它没有laravel-admin后台那么庞大复杂。个人也比较喜欢adminlte的样式设计，感觉要比layuiAdmin好看很多。
- [laravel](https://laravel.com).
- [adminLte](https://adminlte.io/).
